## UI Available Scripts
[localhost:3000](https://localhost:3000)

## Node version
```bash
    12.22.11
```

### Before you run the UI
```bash
    cd pizza4p-ui
    npm i
```

### Running the development server.

```bash
    npm run dev
```


## Server Available Scripts
[localhost:3001](https://localhost:3001)

### Before you run the Server
```bash
    cd pizza4p-api
    npm i
```

### Running the development server.

```bash
    npm start
```
